# toggle sneak mod for MC 1.9.4

### purpose 

Key press, you sneak, key press, you walk.   
No finger on Lshift pain.

### download

1.9.4 - [here](https://github.com/tkgwku/tkgw-toggle-sneak/releases/tag/1.9.4-1)

### instruction

1. install forge
2. put this in "mods" folder
3. run minecraft
